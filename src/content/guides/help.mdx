# Getting Help

If you run into a problem or have a question, there are two ways to reach us:

## File a GitHub Issue

For bug reports, feature requests, or general questions, open an issue on the [iron-proxy GitHub repository](https://github.com/ironsh/iron-proxy/issues). When filing an issue, please include:

- Your iron-proxy version (run `iron-proxy --version`).
- Whether you are using a Docker image or a standalone binary.
- Details about your deployment: cloud provider, orchestration tool, and network configuration.
- Steps to reproduce the problem, along with any relevant logs or CLI output.

## Email Support

You can also reach us at [support@iron.sh](mailto:support@iron.sh). Please include the same information listed above so we can help you as quickly as possible.
