# Getting Help

If you run into a problem or have a question, there are two ways to reach us:

## File a GitHub Issue

For bug reports, feature requests, or general questions, open an issue on the [Irons GitHub repository](https://github.com/ironsh/irons/issues). Include as much detail as you can — sandbox IDs, CLI output, and steps to reproduce are all helpful.

## Email Support

For anything else, send us an email at [support@iron.sh](mailto:support@iron.sh) and we'll get back to you as soon as we can.
