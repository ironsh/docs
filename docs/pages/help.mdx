# Getting Help

Two ways to reach us: GitHub issues for bug reports and feature requests, email for everything else.

## File a GitHub Issue

Open an issue on the [iron-proxy GitHub repository](https://github.com/ironsh/iron-proxy/issues). Include:

- Your iron-proxy version (run `iron-proxy --version`).
- Whether you are using a Docker image or a standalone binary.
- Details about your deployment: cloud provider, orchestration tool, and network configuration.
- Steps to reproduce the problem, along with any relevant logs or CLI output.

## Email Support

Email [support@iron.sh](mailto:support@iron.sh) with the same information listed above.
